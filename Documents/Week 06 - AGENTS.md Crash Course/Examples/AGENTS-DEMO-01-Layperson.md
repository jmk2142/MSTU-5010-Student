# AGENTS-DEMO-01-Layperson.md

This is a teaching demo file, not a live repo instruction file.
Use this filename for reference examples so tools do not treat it as active `AGENTS.md` policy.

## Purpose
Show AGENTS.md in plain language for someone new to agents.

## Demo policy
- Work only in the current folder.
- Make one small edit only.
- Do not delete files.
- Do not run git commands.
- At the end, explain what changed in 3 short bullets:
  - what you changed
  - why you changed it
  - what you did not change

## Example class task
Fix one typo in a README and add one sentence clarifying the project goal.
