## Reporting Pattern Notes

This scaffold is intentionally opinionated. It demonstrates that if students specify:

- status labels
- evidence fields
- exact tables
- final state format
- expected output sections

the agent is more likely to return a useful, auditable report instead of a vague summary.

## Reuse Note

If `TEST-RESULTS.md` is missing, create a fresh copy from this template structure before running the suite.
If `TEST-RESULTS.md` already exists, overwrite it for the new run and increment the run iteration at the top of the report.
